# Karma unit testing ES6 javascript with Webpack, babel #


## Building, running test and starting ##

```bash
git clone https://github.com/andrepestana/karma-es6-babel-webpack_boilerplate_boilerplate
cd karma-es6-babel-webpack
npm i

npm i -g karma-cli
npm run tests

npm install -g npx
npm run build

npm run start
```
